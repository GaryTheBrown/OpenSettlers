Open Settlers
=================

Synopsis
--------

A open source Game Engine to run the Settlers Series (1-4 Planned)

Usage
-----
ON Linux
./OpenSettlers

To Extract
./OpenSettlers -e [folder/file]

Installation
------------
mkdir build
cd build
cmake ../
make
