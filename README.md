SettlersExtractor
=================

Synopsis
--------

A open source project to extract graphics and sound from the Settlers Series

Usage
-----
ON Linux
./SettlersExtractor [Folder/File]

Installation
------------

IMPORT INTO ECLIPSE AND COMPILE.
A AUTOMAKE SYSTEM WILL BE ADDED AT SOME POINT FOR BUILDING
